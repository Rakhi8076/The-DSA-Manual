# 🚀 The DSA Manual

The DSA Manual is a full-stack DSA practice platform designed to help students prepare for coding interviews through structured learning paths, progress tracking, and AI-powered guidance.

## ✨ Features

- 📚 Practice questions from:
  - Striver A2Z Sheet
  - Love Babbar DSA Sheet
  - Apna College DSA Sheet
  - Merged DSA Sheet

- 📈 Track solved and unsolved questions
- 🔥 GitHub-style coding activity heatmap
- 🤖 AlgoShee AI Assistant for DSA guidance
- 💡 AI-generated personalized learning insights
- 🔐 JWT-based Authentication
- 📊 Dashboard with topic-wise and sheet-wise analytics
